
SISKA LARAVEL PROJECT (SCAFFOLD)

Fitur:
- Login
- Dashboard
- Surat Masuk
- Surat Keluar
- Disposisi

Cara pakai:

1 Install Laravel
composer create-project laravel/laravel siska

2 Copy file scaffold ini ke project Laravel

3 Jalankan migration
php artisan migrate

4 Jalankan server
php artisan serve
